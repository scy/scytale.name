TOFU-Grillen für Anfänger

Wie man angenehm lesbare Nachrichten schreibt

zurück: Aufbau einer Mailhoch: Technischesweiter: Mails im falschen Thread

Threads intern

  • Jede Message eindeutig identifiziert über Message-ID-Header
  • Beim Antworten auf eine Nachricht wird In-Reply-To-Header mit ID der Ursprungsnachricht vom Mailclient eingefügt
  • Außerdem (bei guten Clients) References-Header mit allen IDs aus dem Thread
